Modelo Osi

O modelo OSI é um padrão para os protocolos de rede. Protocolos nada mais são do que regras de comunicação usadas para conectar dois ou mais computadores.

TCP/IP

São dois protocolos garantem que pacotes de informações cheguem a seu destino de forma correta e segura.

dns

São dois protocolos garantem que pacotes de informações cheguem a seu destino de forma correta e segura.

UDP

é mais simples e não exige uma conexão, nele os serviços de recuperação e de verificação não são necessários.

DHCP

é o protocolo que automaticamente fornece um host com o seu endereço de IP.

FTP

é usado para a transferência de arquivos à distância, dentro do conjunto TCP/IP. Ele é um protocolo para o upload e download de arquivos.

http/https

é um protocolo de comunicação ou seja ele defini regras e padrões para que uma conexão e troca de dados entre sistemas computacionais seja estabelecida. De um lado, um navegador requisita um determinado dado, e do outro, um computador (ou servidor) retorna a informação desejada

Webserver

É um sistema computacional que concede acesso a conteúdos e aplicações da internet.
